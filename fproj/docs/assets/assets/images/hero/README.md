# Hero Images Directory

Place hero/background images for featured content in this directory.

## Recommended Specifications:
- **Size**: 1920x1080px (16:9 aspect ratio)
- **Format**: JPG, PNG, or WebP
- **File naming**: Use descriptive names matching the poster images

## Example Files:
- dark_knight.jpg
- stranger_things.jpg
- inception.jpg

These images are used in the hero section on the home page and details pages.

Update the `heroImageUrl` paths in `lib/models/content.dart` to match your actual image filenames.
