# Poster Images Directory

Place movie and TV show poster images in this directory.

## Recommended Specifications:
- **Size**: 300x450px (2:3 aspect ratio)
- **Format**: JPG, PNG, or WebP
- **File naming**: Use descriptive names like `dark_knight.jpg`, `stranger_things.jpg`, etc.

## Example Files:
- dark_knight.jpg
- stranger_things.jpg
- inception.jpg
- crown.jpg
- interstellar.jpg
- breaking_bad.jpg
- shawshank.jpg
- got.jpg
- john_wick.jpg
- mad_max.jpg
- office.jpg
- grand_budapest.jpg

Update the image paths in `lib/models/content.dart` to match your actual image filenames.
